Defining Correspondence Types Definiendo los tipos de correspondencia Correspondence

Se define un tipo de correspondencia distinto para cada una de las tres cartas — Balance Confirmation, Balance Notification, Balance Request — cada uno con su propio texto base.

Setting Tolerances for Balance Differences Definiendo tolerancias para diferencias de saldo Business Transactions
La diferencia de €80 que vio Jake con Tireless Tires — ¿cuánto es "una diferencia real"?
Se define un margen de tolerancia por debajo del cual una diferencia informada por el proveedor se considera menor y no dispara una investigación formal — por encima de ese margen, sí.
