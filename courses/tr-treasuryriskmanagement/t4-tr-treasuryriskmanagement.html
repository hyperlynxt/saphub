Configuring Parallel Valuation Areas (IFRS / GAAP) Configurando áreas paralelas de valuación Hedge Accounting
Que el sistema pueda contabilizar bajo IFRS o GAAP no es automático — hay que definir ambas áreas.
Concepto clave: se configuran áreas de valuación paralelas — una para IFRS, otra para GAAP si aplica — cada una con sus propias reglas de reconocimiento, para que el mismo hedge pueda valuarse bajo ambos estándares sin duplicar el dato maestro.

Defining the Effectiveness Test Method Definiendo el método de prueba de efectividad Hedge Accounting
